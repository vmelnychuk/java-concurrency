    - Callable vs Runable (new Callable <-> new Runnable)
    - Futures

        - get
            - null
            - wait until done
    - ThreadLocal
    - volatile
    - synchronized
    - lock
    - Lock vs synchronized
    - Semaphores
    - ExecutePool
